How to enable camera access in Telegram:

  1. Tap the "Allow Camera Access" button above
  2. If prompted, allow camera permissions
  3. If no prompt appears, follow these steps:
  4. Tap the menu (⋮) in the top right
  5. Go to "Settings" → "Permissions"
  6. Enable camera access for this site
  7. Refresh the page and try again
